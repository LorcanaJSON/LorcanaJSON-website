2025-08-26 2.2.1

  • Format addition: Because of the two new rarities, the fields epicId and iconicId are added. They work the same as enchantedId
  • Format addition: Instead of creating the equivalent of nonEnchantedId for Epic- and Iconic-rarity cards, a new consolidated field has been added: baseId. This field points to the base card within the normal numbering from Promo cards and from cards with Epic, Enchanted, or Iconic rarity
  • Warning: Because the aforementioned baseId consolidates them, nonEnchantedId and nonPromoId will be removed in the next update on September 5th
  • Technical note: Previously the zipfiles were compressed with the LZMA-algorithm. From this release on, the DEFLATE-algorithm will be used. The zipfiles are slightly larger, but DEFLATE has larger compatibility. (If this doesn't mean anything to you, it probably doesn't affect you)
  • Corrected the date when sets 1 to 4 are rotated out of Core Constructed from "2025-09-19" to "2025-09-05", the date when Set 9 fully releases
  • The 'Cold' foil type was renamed to 'Silver'

2025-07-25 2.2.0

  • Format addition: allowedInFormats is a new dictionary field in sets and cards that contains data per format ("Core" and "Infinity" for now) saying whether the set or card is currently allowed in that format. If it's allowed until a certain date, that info is also included. If a card is banned in that format, that is included in this format dictionary. See the allowedInFormats explanation for more details
  • Breaking format change: Because per-format ban data is now contained in allowedInFormats as mentioned above, the separate bannedSince card field has been removed
  • Format addition: allowedInTournamentsFromDate card field contains a date string indicating when this set or card is/was allowed to be used in official tournaments
  • Format addition: Since cards are getting reprinted, two fields have been added: reprintedAsIds is a list of card IDs that are reprints of the current card; and reprintOfId is the ID of the card that this is a reprint of. Neither field exists if they're not relevant. Currently only two cards have been reprinted:
    • Capitan Uncino - Duellante Aggressivo: ID 174 is the original print, and ID 1850 is the reprint
    • Cucciolo di Dalmata - Scodinzolante: ID 436 is the original print, and ID 1702 is the reprint. Note: This card has variants, but only the first variant is used for the reprint fields
